create ingredients for new element

for new html element

  • now create a container for element
              
                  const child = document.createElement("div")

  • add one or multiple className to the element
              
                  child.classList.add("className1", "className2")

  • add text content to the element
              
                  child.textContent = "our message";

  • [alternatively] add html content to the element 
              
                  child.innerHTML = `hardcode full html codes`

add element and show

adding element and show it in document

  • add the child element as a last child 
              
                  parent.append(child)
  • add the child element as a first child 
              
                  parent.prepend(child)
  • just before parent as sibling position 
              
                  parent.before(child)
  • just after parent as sibling position 
              
                  parent.after(child)
