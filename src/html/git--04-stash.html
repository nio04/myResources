stash : information

stash changes : important information

git stash hide our works from staged & unstaged area. so that we can switch to another branch, without having to worry about conflict warning or unstaged works follow to another branch

when we have works in one branch and we want to switch to another branch but we dont want to commit our current works (without commiting, git wont let switch!), then we stash the changes

stash : operation

stash o/p

              
                // to stash changes
                git stash
                // to un-stash changes
                git stash pop

git wont allow to switch branch, when we have un-commited changes in our current branch. if we still want to switch branch, we stash them (because we dont want to commit our current works)
