array destructure

Basic Array Destructuring

              
                let fruits = ['Apple', 'Banana'];
                let [fruit1, fruit2] = fruits;

Default Values

              
                let [fruit1, fruit2 = 'Mango'] = ['Apple'];

When the unpacked value is undefined, a default value can be assigned

Skipping Items

              
                 let [fruit1, , fruit3] = ['Apple', 'Banana', 'Grapes'];

we can skip values that are not needed... by using commas

Swapping Variables

              
              let a = 1, b = 2;
              [b, a] = [a, b];

Rest Pattern

              
                let [fruit1, ...otherFruits] = ['Apple', 'Banana', 'Grapes', 'Mango'];

Use rest pattern to capture the rest of the elements into a new array.

Nested Array Destructuring

              
                let colors = ['red', ['green', 'lightgreen'], 'blue'];
                let [color1, [color2]] = colors;

Destructuring Function Parameters

              
              function describeFruit([fruit, color]) {
                console.log(`${fruit} is ${color}.`);
              }

              describeFruit(['Mango', 'yellow']); // Mango is yellow.

Use with Iterables

              
                let [a, b, c] = 'abc';

Destructuring with Strings

              
                let [char1, char2] = 'hi';

object destructure

Basic Destructuring

              
                const person = {
                  name: 'John',
                  age: 30
                };

                // Destructure the object
                const { name, age } = person;

can extract multiple properties from an object and assign them to variables with the same names as the properties

Renaming Variables

              
                const person = {
                  name: 'John',
                  age: 30
                };

                // Destructuring with different variable names
                const { name: personName, age: personAge } = person;

can extract multiple properties from an Assign properties to variables with different names, we can do so by providing a new variable name

Default Values

              
                const person = {
                  name: 'John',
                  age: 30
                };

                // Default values
                const { gender = 'Unknown' } = person;

can assign default values that will be used if the property is not found on the object.

Nested Object Destructuring

              
                const person = {
                  name: 'John',
                  age: 30,
                  parent: {
                    father: 'Robert'
                }
                };

                // Nested object destructuring
                const {
                name,
                age,
                parent: { father }
                } = person;

Function Parameter Destructuring

              
                function greet({ name, age }) {
                  console.log(`Hello, my name is ${name} and I am ${age} years old.`);
                }

                const person = {
                  name: 'John',
                  age: 30
                };

                greet(person);

Destructure objects right in function parameters. This can be especially useful when dealing with options objects.

REST Pattern with Destructuring

              
                const person = {
                  name: 'John',
                  age: 30,
                  gender: 'Male'
                };

                // REST pattern in destructuring
                const { name, ...rest } = person;
                console.log(name); // John
                console.log(rest); // { age : 30, gender : male}

Rest properties collect the remaining enumerable property keys that are not already picked off by the destructuring pattern.
