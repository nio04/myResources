adding css class

adding css stylesheet class

removing css class

removing existing css stylesheet class

              
                const element = document.querySelector("elementClassName")
                element.classList.remove("cssClassName")

toggle css class

toggle css classes

              
                const element = document.querySelector("elementClassName")
                element.classList.toggle("cssClassName")
toggle css classes.
if specifies class exist, then remove it, but if it does not exist then add it

checking class existance

              
                const element = document.querySelector("elementClassName")
                element.classList.contains("cssClassName")
return a boolean value, for specific class existance
