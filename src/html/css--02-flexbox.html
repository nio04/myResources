flex-direction

flex-direction row[default] | row-reverse | column | column-reverse flex-direction controls flex direction flow. defaultly, its row. but when its set to column, then the main-axis become align-content and cross-axis become justify-content

flex wrap

flex-wrap nowrap [default] | wrap | wrap-reverse flex-wrap defines contents wrap behavior

justify-content

justify-content flex-start [default] | flex-end | start | end | left | right | center | space-between | space-around | space-evenly justify-content defines content flow on main-axis or horizontal side

align-items

align-items stretch [default] | flex-start | flex-end | start | self-start | end | self-end | center | baseline align-items defines content flow on cross-axis or verticle side

align-content

align-content normal [default] | flex-start | flex-end | start | end | space-between | space-around | space-evenly | stretch align-content this property only applied, if the contents in multi-line. works in cross axis (y) - between row gap

gap

gap [row gap] [column gap] gap this property set gap between children items

children items
flex-grow

flex-grow [positive integer] flex-grow default 0. give permission to particular child to have increase its size according to the flex-grow property

flex-shrink

flex-shrink [positive integer] flex-shrink default 0. give permission to particular child to have decrease its size according to the flex-shrink property

flex-basis

flex-basis [positive integer] flex-basis defines the default size of an element before the remaining space is distributed

flex

flex [flex-grow] [flex-shrink] [flex-basis] flex flex is shorthand for flex-grow, flex-shrink, flex-basis

order

order [positive integer] order default is probably all to 0. set any higher positive numbers to mess up the order system

align-self

align-self stretch [default] | flex-start | flex-end | start | self-start | end | self-end | center | baseline align-self override parent container's align-items property for per selected children
