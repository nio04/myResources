parent container

flex-direction

controls flex direction flow. defaultly, its row. but when its set to column, then the main-axis become align-content and cross-axis become justify-content

flex-wrap

defines contents wrap behavior

justify-content

defines content flow on main-axis or horizontal side

align-items

defines content flow on cross-axis or verticle side

align-content

this property only applied, if the contents in multi-line. works in cross axis (y) - between row gap

gap

this property set gap between children items

children items

flex-grow

default 0. give permission to particular child to have increase its size according to the flex-grow property

flex-shrink

default 0. give permission to particular child to have decrease its size according to the flex-shrink property

flex-basis

defines the default size of an element before the remaining space is distributed

flex

flex is shorthand for flex-grow, flex-shrink, flex-basis

order

default is probably all to 0. set any higher positive numbers to mess up the order system

align-self

override parent container's align-items property for per selected children
