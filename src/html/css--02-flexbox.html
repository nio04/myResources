parent container

flex-direction

              
                flex-direction : row[default] | row-reverse | column | column-reverse;

controls direction flow. when its set to column, then align-items -> main-axis and justify-content -> cross axis

flex-wrap

              
                flex-wrap : nowrap [default] | wrap | wrap-reverse;

if flex-wrap property is set to wrap then, when the children width exceeded parent container width, the children items will be in new line

justify-content

              
                justify-content : flex-start [default] | flex-end | start | end | left| right | center | space-between | space-around | space-evenly

defines children alignment on main-axis (horizontal)

align-items

              
                align-items : stretch [default] | flex-start | flex-end | start | self-start | end | self-end | center | baseline;

defines children alignment on cross-axis (verticle)

align-content

              
                align-content : normal [default] | flex-start | flex-end | start | end | space-between | space-around | space-evenly | stretch

controls the alignment of children in cross axis, when there is enough space in parent container

gap

              
                gap : [row gap] [column gap]

this property set gap between children items. apply gap only between children items, not the outer of the children items

children items

flex-grow

              
                flex-grow : [positive integer]

default is 0. add the ability to grow as much as possible within the container. it calculate the width of specific element relation to other items width.

suppose, first item have flex-grow: 1; the second item have flex-grow: 2; so, the second item will be twice larger relation to first one.

flex: 1; means, flex-grow: 1;

flex-shrink

              
                flex-shrink : [positive integer]

default is 1. when a specific element is allowed to shrink down, then when viewport width are decreasing, that specific element also keeps decreasing its width.

but! if we dont want it to shrink down its specific width, then we specify its flex-shrink : 0; then its width never get shrink down

flex-basis

              
                flex-basis : [positive integer]

default is auto. it define the width of the specific element

flex

              
                flex : [flex-grow] [flex-shrink] [flex-basis]

flex is shorthand for flex-grow, flex-shrink, flex-basis

order

              
                order : [positive integer]

default value is 0 for all the child items

align-self

              
                  align-self : stretch [default] | flex-start | flex-end | start | self-start | end | self-end | center | baseline

override parent container's align-items property for specific children
