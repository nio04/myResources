diff : operation
unstaged :: last commit 
              
              git diff
compare codes from last commit to current unstaged works
staged :: last commit 
              
              git diff HEAD
compare codes from last commit to current staged and unstaged works
see changes in a specific file 
              
              git diff HEAD [file path/file name]
comapre code from last commit to staged and unstaged area for a specific file
diff for branches 
              
              git diff [first branch name]..[second branch name]
ORDER MATTERS!!! [first branch] changes will have (-) sign && [second branch] changes fill will have (+) sign.

[first branch] (+) means, from where to look up for changes [second branch] (-) is kinda like, where we test the difference
diff for commit 
              
              git diff [first commit hash]..[second commit hash]
ORDER MATTERS!!! [first commit] will be like a book, where we will compare our changes to another commit
