spread operator

information about spread operator

The spread operator allows an iterable, such as an array, to be expanded into individual elements.

This operator is commonly used to make copies of arrays, merge arrays, or pass elements of an array as arguments to a function

Combining Arrays

              
                let fruits = ['apple', 'banana'];
                let berries = ['strawberry', 'raspberry'];
                let combined = [...fruits, ...berries];
                console.log(combined); // Output: ["apple", "banana", "strawberry", "raspberry"]

Spread syntax can combine multiple arrays into one.

Concatenating Arrays

              
                let numbers1 = [1, 2, 3];
                let numbers2 = [4, 5, 6];
                let concatenated = [...numbers1, ...numbers2];
                console.log(concatenated); // Output: [1, 2, 3, 4, 5, 6]

Copying Arrays

              
                  let original = [1, 2, 3];
                  let copy = [...original];
                  console.log(copy); // Output: [1, 2, 3]

Spreading Elements on Function Calls

              
                function sum(x, y, z) {
                  return x + y + z;
                }

                let numbers = [1, 2, 3];
                console.log(sum(...numbers)); // Output: 6

Spreading Strings

              
              let str = "hello";
              let chars = [...str];
              console.log(chars); // Output: ["h", "e", "l", "l", "o"]

Object Literals

              
                let obj1 = { foo: 'bar', x: 42 };
                let obj2 = { foo: 'baz', y: 13 };
                let mergedObj = { ...obj1, ...obj2 };
                console.log(mergedObj); // Output: { foo: "baz", x: 42, y: 13 }
                // Note that obj2's foo overwrites obj1's foo.

                let originalObj = { a: 1, b: 2 };
                let clonedObj = { ...originalObj };
                console.log(clonedObj); // Output: { a: 1, b: 2 }

rest paramter

information about rest parameter

Rest parameter syntax in JavaScript is used to represent an indefinite number of arguments as an array.

Typically used in function parameters and destructuring assignments.

Function Arguments

              
                function sum(...numbers) {
                  return numbers.reduce((acc, current) => acc + current, 0);
                }

                console.log(sum(1, 2)); // 3
                console.log(sum(1, 2, 3, 4, 5)); // 15

Rest syntax allows you to create functions that accept any number of arguments.

Destructuring Arrays

              
                const [first, ...restOfItems] = [10, 20, 30, 40];

                console.log(first); // 10
                console.log(restOfItems);  // [20, 30, 40]

Rest syntax can gather the rest of the elements into an array when destructuring.

Destructuring Objects

              
              const { a, ...restOfProps } = { a: 1, b: 2, c: 3, d: 4 };

              console.log(a); // 1
              console.log(restOfProps); // {b: 2, c: 3, d: 4}

can collect remaining enumerable properties of an object.

Combine Arrays

              
              const fruits = ['apple', 'banana'];
              const moreFruits = ['orange', 'grape', ...fruits];

              console.log(moreFruits); // ['orange', 'grape', 'apple', 'banana']

Rest syntax can be a clean method to combine arrays
