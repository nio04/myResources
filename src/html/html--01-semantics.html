semantic tags

section

It represents the section of the document.

header

represents a container for introductory content or a set of navigational links.

nav

defines a set of navigation links.

main

the main page content and should be unique

article

specifies independent, self-contained content.

details

specifies the tag for additional details.

mark

specifies the text that is highlighted.

summary

specifies a header for the details element.

aside

defines some content aside from the content it is placed in (like a sidebar).

figure

wrap img element

figcaption

adding a caption to the image

footer

defines a footer for a document or section
